# 为Endnote中文用户提供一种简单的使用方式
## 正在开发的功能
- 为本地全文生成引文文件(格式为enw)

## 使用方法
1. `pip install -r requirements.txt`
2. `python main.py`

### 在Windows环境下直接使用的可执行程序
简单打包了一个exe文件（[下载链接](https://github.com/vc5/endnote-cn/releases/download/0.0.1/endnote-cn.exe)），下载后点击使用即可
